#  Sentiment Analysis on Social Media Data

##  Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards different topics or brands.

---

##  Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- TextBlob (NLP)
- Excel

---

##  Dataset
The dataset contains social media comments stored in an Excel file (`Task 4.xlsx`).  
It includes text data collected from different platforms like Twitter, Instagram, and Facebook.

---

##  Steps Performed
1. Data Collection from Excel file  
2. Data Cleaning (handling missing values)  
3. Sentiment Analysis using TextBlob  
4. Classification into:
   - Positive   
   - Negative   
   - Neutral  
5. Data Visualization using charts  

---

##  Output
The following outputs are generated:

- **Bar Chart** – Shows count of each sentiment  
- **Pie Chart** – Shows percentage distribution  
- **CSV File** – Contains analyzed results
